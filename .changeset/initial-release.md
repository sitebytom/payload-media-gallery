Initial release of the plugin features:
- **Justified, Grid and List Layouts**: Custom layouts for media collections (Justified as default).
- **Lightbox**: Full-screen viewer for images, video playback, and audio files.
- **Focal Point Support**: Support for Payload focal point cropping.
- **Document Handling**: Graceful fallbacks and download links for non-media files.
- **Video Previews**: Auto-playing video previews on hover.
- **Quick Edit**: Integration with native Payload drawers for fast metadata updates.
- **Bulk Selection**: Enhanced multi-selection with keyboard modifier support.
- **Payload-Native UI**: Seamlessly integrated into the Payload admin dashboard.
